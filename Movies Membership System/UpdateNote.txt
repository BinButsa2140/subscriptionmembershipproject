ต้า 24/9/2023
    เพิ่มคลาส StandardUserUi และ PremiumUserUi เพื่อที่จะใช้ทำ Interface

    แก้ไขให้ การแสดงข้อมูลในแต่ละคนมีคลาสแยกเป็นของตัวเอง เพื่อที่จะทำให้โปรเจคมี Interface ที่มากและดูดีขึ้น 
    โดยจะแบ่งออกเป็น 3 คลาส 
    -StandardUserUi
    -PremiumUserUi
    -UserInformationForAdmin (อันนั้นแก้ไขชื่อและรายละเอียดในคลาส)
    -โดยที่คลาสนั้นจะทำการ Implement มาจากคลาส Userinterface.java(ยกเว้นของAdmin) แต่จะมีความแตกต่างกันออกไป
    -สิ่งที่ต้องทำที่เหลือคือเพิ่มเงื่อนไขตรงช่วง login เพื่อดูว่าเป็น User ชนิดไหน จะได้เปิดพาแนลนั้นๆถูก (StandardUserUi PremiumUserUi)
    -ทั้งสองคลาสที่เป็น Ui ของ user ถูกเทสแล้ว ใช้งานได้ปกติ เหลือแค่นำไปใช้งาน 

    ปัญหาที่พบ
    -บางครั้งล็อกอินด้วยรหัสและUserที่มีอยู่ในไฟล์ แต่ไม่สามารถเข้าได้อยู่ดี(เกิดขึ้นบางครั้ง และบางบัญชี)คาดว่าน่าจะเป็นที่พิมพ์เล็กพิมพ์ใหญ่
    ลองเทสโปรแกรมดู
